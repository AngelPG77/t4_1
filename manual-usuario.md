## Descripción General de la Aplicación

### ¿Qué es la Aplicación Bar?

La Aplicación Bar es un sistema de gestión de pedidos moderno que permite:

-  **Crear pedidos** de forma rápida y organizada
-  **Seleccionar productos** de un catálogo completo
-  **Ver resumen detallado** de pedidos
-  **Calcular totales** automáticamente
-  **Visualizar historial** de pedidos

### Pantalla Principal

Cuando abres la aplicación, la **Pantalla Principal (Home)** muestra:

- **Lista de pedidos activos:** Todos los pedidos que están en proceso
- **Información de cada pedido:** Mesa, cantidad de productos, precio total
- **Botón para crear nuevo pedido:** Acceso rápido para empezar una nueva comanda
- **Información adicional:** Detalles de todos los artículos en cada pedido

### Requisitos Mínimos

- Sistema operativo: Android 5.0+
- Espacio disponible: 50 MB
- Conexión: Opcional (funciona sin internet)

## Primeros Pasos

### 1. Iniciar la Aplicación

Abre la aplicación desde tu dispositivo. Verás la pantalla principal con el listado de pedidos.

### 2. Crear Tu Primer Pedido

Toca el botón **"Nuevo Pedido"** en la pantalla principal para comenzar.

### 3. Indicar la Mesa

En la pantalla de creación:
- Ingresa el **nombre o número de la mesa** en el campo "Mesa"
- Ejemplo: "Mesa 1", "Barra", "Terraza 5"

### 4. Añadir Productos

Toca el botón **"Productos"** para abrir el catálogo y seleccionar los artículos que deseas agregar.

### 5. Guardar el Pedido

Una vez hayas añadido todos los productos:
- Toca **"GUARDAR PEDIDO"** para confirmar
- Verás un mensaje de confirmación
- El pedido aparecerá en la pantalla principal

## Funcionalidades Principales

### 1. Crear Nuevo Pedido

**¿Qué hace?**
Permite iniciar un nuevo pedido indicando la mesa y seleccionando productos.

**Pasos:**
1. Desde la pantalla principal, toca el botón "Nuevo Pedido"
2. Completa el campo "Mesa" con un nombre o número
3. Toca "Productos" para seleccionar artículos
4. Toca "GUARDAR PEDIDO" cuando hayas terminado

**Nota:** No puedes guardar sin mesa y sin al menos 1 producto.

---

### 2. Seleccionar Productos

**¿Qué hace?**
Muestra el catálogo completo de productos disponibles en tu bar.

**Cómo usar:**
- **Para añadir:** Toca el icono + junto al producto
- **Para aumentar cantidad:** Toca + nuevamente (aparece un contador)
- **Para reducir cantidad:** Toca el icono - 
- **Para ver cambios:** Los productos seleccionados se resaltan en color

**Catálogo disponible:**
- Comidas: Burgers, Pizzas, Sopas, etc.
- Bebidas: Refrescos, Agua, Café
- Postres: Tartas

---

### 3. Resumen de Pedido

**¿Qué hace?**
Muestra un resumen completo del pedido con detalles y totales.

**Información mostrada:**
- Mesa del pedido
- Listado de todos los productos
- Cantidad de cada artículo
- Precio unitario y total por artículo
- **Total a pagar**

**Acceso:**
Desde la pantalla de creación, toca "Resumen" (solo disponible si tienes mesa + productos)

---

### 4. Gestionar Pedido

**Acciones disponibles:**

| Acción | Descripción |
|--------|-------------|
| **Modificar Mesa** | Cambia el nombre de la mesa antes de guardar |
| **Añadir Productos** | Toca "Productos" para añadir más artículos |
| **Cambiar Cantidades** | Modifica las cantidades en la pantalla de productos |
| **Limpiar Todo** | Borra todos los datos del pedido (aparece confirmación) |
| **Ver Resumen** | Muestra detalles completos antes de guardar |

## Preguntas Frecuentes

### P: ¿Puedo editar un pedido después de guardarlo?
**R:** En la versión actual, los pedidos guardados se muestran en la pantalla principal. Una vez guardados no se podrán editar.
### P: ¿Qué hago si cometo un error al crear el pedido?
**R:** Antes de guardar, puedes:
- Cambiar la mesa usando el campo "Mesa"
- Usar "Productos" para añadir o quitar artículos
- Usar "Limpiar todo" para empezar de nuevo

### P: ¿Cómo calcula el total?
**R:** El total se calcula automáticamente multiplicando el precio de cada producto por su cantidad. Puedes ver el desglose en la pantalla de "Resumen".

### P: ¿Puedo añadir cantidades grandes de un producto?
**R:** Sí. Toca el botón + tantas veces como necesites. El contador mostrado indica la cantidad actual. No nos hacemos responsables de que luego no te acabes la comida (recomendamos sopa de patata).

### P: ¿Es obligatorio poner nombre de mesa?
**R:** Sí. El sistema requiere una mesa y al menos un producto para guardar. Esto asegura que todos los pedidos estén identificados.

### P: ¿Dónde veo el historial de pedidos?
**R:** En la pantalla principal aparecen todos los pedidos activos/guardados con información completa de mesa y totales.

### P: ¿Qué sucede si cierro la aplicación sin guardar?
**R:** Si cierras sin guardar, los datos del pedido en progreso se perderán. Asegúrate de tocar "GUARDAR PEDIDO" antes de cerrar.

### P: ¿Puedo usar la aplicación sin conexión a internet?
**R:** Sí. La aplicación funciona completamente sin conexión. Los datos se guardan localmente en tu dispositivo.
